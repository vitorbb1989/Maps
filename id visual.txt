IDENTIDADE VISUAL - Sistema Completo
Visão Geral
Este documento define o design system completo da AntropIA: cores, tipografia, espaçamento, componentes e padrões visuais.

Tudo que é visual deve seguir estas especificações.

Filosofia Visual
Apple-like
Nosso design é inspirado na abordagem Apple:

Silencioso, não grita
Funcional, a forma serve a função
Elegante, através da simplicidade
Consistente, padrões previsíveis
Durável, não segue tendências passageiras
Princípios
Menos é sempre mais
Funciona bem em preto e branco
Espaçamento generoso
Hierarquia clara
Sem elementos decorativos
Sistema de Cores
Filosofia: 80/15/5
80%: Base neutra (brancos, cinzas, pretos)
15%: Accent primário (ação, interação)
5%: Cores funcionais (estado, feedback)
Base Neutra (80% da interface)
Fundos:

Primary: #FFFFFF - Fundos principais, cards
Secondary: #F6F4F0 - PDFs, slides, leitura longa
Tertiary: #FAFAFA - Fundos alternativos
Textos:

Primary: #1A1A1A - Texto principal
Secondary: #6B6B6B - Texto secundário, labels
Tertiary: #9CA3AF - Texto terciário, placeholders
Bordas:

Default: rgba(0, 0, 0, 0.1)
Hover: rgba(0, 0, 0, 0.2)
Focus: rgba(26, 115, 232, 0.4)
Accent Primário (15%)
Ação e Interação:

Primary Hover: #1557B0 - Estado hover
Primary Active: #0D3C7A - Estado active
Primary BG: rgba(26, 115, 232, 0.08) - Fundos accent
Primary Border: rgba(26, 115, 232, 0.2) - Bordas accent

Regras de Uso
✅ Fazer:

Usar tokens semânticos (--primary, não #1A73E8)
Manter 80% da interface neutra
Usar accent apenas para ação
Cores funcionais apenas para estado
❌ Não fazer:

Misturar HEX direto no código
Usar cores funcionais decorativamente
Criar novas cores fora do sistema
Usar accent em fundos dominantes
Tipografia
Fonte
Inter (Google Fonts)

Weights: 400 (Regular), 500 (Medium), 600 (Semibold)
Fallback: -apple-system, BlinkMacSystemFont, system-ui, sans-serif
Escala Tipográfica
Display/Titles:

4XL: 36px / 40px (letter-spacing: -0.015em)
3XL: 30px / 36px (letter-spacing: -0.015em)
2XL: 24px / 32px (letter-spacing: -0.015em)
Headings:

XL: 20px / 28px
LG: 18px / 28px
Body:

Base: 16px / 24px (texto principal)
SM: 14px / 20px (labels, UI)
XS: 12px / 16px (captions, metadata)
Regras de Uso
Hierarquia:

1 tamanho de display por página
Máximo 3 níveis de hierarquia visível
Diferença mínima entre níveis: 4px
Weights:

Regular (400): Corpo de texto, parágrafos
Medium (500): Labels, navigation
Semibold (600): Títulos, ênfase
Letter-spacing:

Tight (-0.015em): Displays e títulos grandes
Normal (0): Corpo de texto
✅ Fazer:

Usar escala definida
Semibold apenas em títulos
Letra spacing consistente
❌ Não fazer:

Inventar tamanhos fora da escala
Bold em corpo de texto
Letter-spacing aleatório

Sombras
Escala
SM: 0 1px 2px rgba(0, 0, 0, 0.05) - Sutil
MD: 0 4px 6px rgba(0, 0, 0, 0.07) - Cards default
LG: 0 10px 15px rgba(0, 0, 0, 0.10) - Cards elevados

Uso
Cards: MD no default, LG no hover
Modais: XL
Dropdowns: LG
Elementos fixos (header): SM
✅ Fazer:

Sombras sutis
Consistência no uso
Sombra aumenta levemente no hover
❌ Não fazer:

Sombras muito pesadas
Múltiplas sombras em um elemento
Sombra sem propósito
Transições
Duração
Fast: 150ms - Micro-interações (hover)
Base: 200ms - Padrão (animações gerais)
Slow: 300ms - Transições complexas
Easing
Padrão: cubic-bezier(0.4, 0, 0.2, 1)
Uso
Hover states: 150ms
Accordion/collapse: 200ms
Page transitions: 300ms
Loading: 200ms
✅ Fazer:

Usar easing padrão
Transições sutis e rápidas
Consistência na duração
❌ Não fazer:

Transições muito longas (> 400ms)
Easing complexo sem necessidade
Animações chamativas

Checklist de Validação
Todo design deve passar por:

[ ] Usa apenas cores do sistema?
[ ] Tipografia segue a escala?
[ ] Espaçamento é múltiplo de 4px?
[ ] Border radius está na escala?
[ ] Sombras são sutis?
[ ] Transições são rápidas (< 300ms)?
[ ] Funciona bem em preto e branco?
[ ] Hierarquia visual é clara?
[ ] Próxima ação é óbvia?
[ ] Nenhum elemento decorativo desnecessário?
Se todos "SIM" → design aprovado. Se algum "NÃO" → ajustar.
